Accessibility Statement

Our commitment to making our website accessible to everyone.

Last updated: March 26, 2026

1. Our Commitment to Accessibility

The Chemical Engineering Society (ChES), IIT Delhi is committed to ensuring that our website is accessible to all users, including those with disabilities. We strive to comply with the Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards and are continuously working to improve accessibility across our digital platforms.

2. Accessibility Features

Our website includes the following accessibility features:

2.1 Navigation and Structure

  • Semantic HTML structure with proper headings hierarchy
  • Descriptive link text and alt attributes for images
  • Consistent navigation across all pages
  • Skip navigation links for keyboard users

2.2 Visual Design

  • High contrast color schemes
  • Dark mode toggle for reduced eye strain
  • Responsive design that works on all screen sizes
  • Readable font sizes and adequate spacing

2.3 Keyboard Navigation

  • All interactive elements are keyboard accessible
  • Logical tab order throughout the site
  • Visible focus indicators
  • Keyboard shortcuts where applicable

2.4 Screen Reader Support

  • Proper ARIA labels and landmarks
  • Descriptive alt text for all images
  • Form labels and error messages
  • Announcement of dynamic content changes

3. Assistive Technologies

Our website is designed to work with:

  • Screen readers (NVDA, JAWS, VoiceOver)
  • Screen magnification software
  • Speech recognition software
  • Alternative input devices
  • Braille displays

4. Content Accessibility

4.1 Text Content

  • All text content meets WCAG color contrast requirements
  • Text can be resized up to 200% without loss of functionality
  • Clear and simple language
  • Consistent terminology throughout the site

4.2 Multimedia Content

  • Descriptive captions for videos
  • Transcripts available for audio content
  • Alt text for all images and graphics
  • Accessible video players with controls

4.3 Forms and Interactive Elements

  • Properly labeled form fields
  • Clear error messages and validation
  • Logical form structure
  • Accessible date pickers and dropdowns

5. Browser and Device Support

Our website is optimized for:

  • Modern web browsers (Chrome, Firefox, Safari, Edge)
  • Mobile devices and tablets
  • Desktop computers
  • Assistive technology combinations

6. Known Limitations

While we strive for full accessibility, some legacy content or third-party embedded content may not fully meet WCAG standards. We are actively working to address these issues. If you encounter any accessibility barriers, please contact us so we can improve our site.

7. Feedback and Contact

We welcome feedback on the accessibility of our website. If you experience any difficulties or have suggestions for improvement, please contact us:

8. Accessibility Resources

For more information about web accessibility:

  • WCAG 2.1 Guidelines
  • Web Accessibility Initiative (WAI)
  • International Association of Accessibility Professionals

9. Continuous Improvement

Accessibility is an ongoing commitment. We regularly audit our website, conduct user testing with people with disabilities, and implement improvements based on feedback and technological advancements.

10. Legal Compliance

ChES is committed to complying with applicable accessibility laws and standards, including but not limited to:

  • WCAG 2.1 Level AA
  • Section 508 of the Rehabilitation Act (where applicable)
  • Information Technology Act, 2000 (India)
  • Rights of Persons with Disabilities Act, 2016 (India)
